LoRA Preflight packaged build

This folder already includes wd-eva02-large-tagger-v3.
No model download is needed after unpacking.

Start:
1. Double-click run_windows.bat
2. Open the URL shown in the console if the browser does not open automatically.

Notes:
- The bundled tag model is in models\wd-eva02-large-tagger-v3.
- Python packages are bundled in wheelhouse for offline install.
- The bundled wheels are for Windows x64 / CPython 3.11.
- If Python is not installed, install Python 3.11 first, then run run_windows.bat again.
