## タイトルのルール

- 形式: `<本文（口語・一文）> #深夜のドカ食い天使ちゃん`
- **絵文字なし・【】囲みなし**（音楽コラボ回のみ例外的に♪絵文字あり）
- チャンネルタグ `#深夜のドカ食い天使ちゃん` を必ず末尾に付ける
- 追加ハッシュタグ（#anime #アニメ 等）をタイトルに足す回もあるが、
  **再生数上位はチャンネルタグのみのシンプル型**

### パターン（再生数実績順）

1. **デッドパン型**（最有力。1.6k〜1.8k views）
   「◯◯を深夜に摂取してハイになるだけの動画」
   「深夜残業で終わった心にひたすらカツカレーを流し込むだけのアニメ」
   → 構文: `<食べ物+状況>を<動詞>だけの動画/アニメ`。自虐・淡々がキモ
2. **ギルティ共感型**（1.4k）
   「深夜にトロトロ卵のカツ丼とかギルティすぎん？」
   → 視聴者への話しかけ。「〜すぎん？」「〜じゃない？」
3. **キャラ独白型**（1.1k〜1.3k）
   「こんな時間にハンバーガーなんて贅沢すぎるかも」
   「今日は会社でなにもしてないけどハンバーガーがこんなに美味いからヨシ！」
   → 天使ちゃんの一人称。仕事ネタ×食で落差を作る
4. ~~掛け算型「深夜×唐揚げ×ドカ食い」~~（0.7k と最弱。使わない）
