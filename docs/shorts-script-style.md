# 脚本・語りのスタイルガイド（30秒ショート用）

> 2026-07-07 作成、同日 **公式脚本（2026.7.1版・7話分）を取得して原文ベースに改訂**。
> 原本: Drive 脚本フォルダ（ガスト唐揚げ丼/辛ラーメン/冷やしまぜそば/生牡蠣/ココイチ/焼肉/背脂二郎）。

## 0. 公式脚本の型（本編。ショートはこれの圧縮形）

全話共通の構造:
1. ◯繁華街(夜): **等身大の悩みモノローグ**（計画性がない/挑戦する気力が出ない/後輩との
   距離感/何者にもなれなかった/成長実感がない）。愚痴ではなく内省。
2. 店の発見: こじつけロジックで食に接続（「熱くなれ→熱い料理」）。「だが、しか〜し！」
3. **アイキャッチ**
4. 店内: 悩みの続き → 着丼「うわぁ〜！」「キッター！」
5. **早口紹介パート（署名芸）**: 「（早口で）今日ドカ食いしていくのは◯◯！ …こいつは
   まさに【命名】だー！」— 必ずオリジナルの命名で締める:
   「旨辛スタミナアトラクション」「食べる夏フェス」「ジャンクビューティー」
   「背脂モンスター」「牡蠣好きの夢を視覚化したオブジェ」
6. 実食レポ（Ｍ）→ **定型句3連**: 「よ〜し…そろそろ…」→「ドカ食いしちゃうぞ♪」→
   完食「ふ〜……ごちそうさまでした♪」
7. ◯繁華街(夜) 帰路: **食べた体験が悩みへの小さな答えのメタファーになる**
   （冷やしまぜそば→「冷たくてマシマシな人生にしてこっと！」）+ 前向き一言

ショートで流用すべき署名要素: 早口命名芸 / 定型句3連 / 悩み→食(行動)→小さな答えの
三段 / モノローグ(Ｍ)主体の独白。

## 1. 天使ちゃんの語りの特徴（実セリフから）

### 文体・口調（公式脚本の原文から）
- モノローグ語尾: 「〜なんだよなぁ」「〜な訳でして」「〜のよな〜♪」「〜だよなあ……」
- **「すぎる」「ギルティ」「背徳感」の多用**: 「助かりすぎんのよな〜」「ギルティすぎんよ…！」
- 転換の決まり文句: 「だが、しか〜し！」「ではでは、いただきます！」
- 感嘆: 「うわぁ〜〜〜！！！」「うっひょ〜い！！！」「キッター！！！」
- **自己正当化・自己ツッコミ**: 「罪な女……」「意外と承認欲求が強めなのかも、私」
  「と、急に悟ってみたり（して）」「ギブミー自制心」
- 突発的キャラ遊び: お嬢様モード（「おどきになってー！」「〜ですわよ！オーホッホ」）、
  謎キャラとの空耳会話（カレー君「ココダァヨ」）
- 語の誤変換的かわいさ: 「広がりゅよ〜♪」
- 敬語とタメ口の混在（気持ちが乗ると崩れる）。宣言→実行（「ドカ食いしちゃうぞ♪」）

### 内容の型
- 理不尽（会社・先輩）→ 小さな回復（食・ぬい・夜）の往復
- 数字と固有名詞で具体化（10枚、2000kcal、横浜家系、深夜1時）
- 最後に**現実がちょっとだけ顔を出す**（ニンニク臭を気にする、明日も仕事）

### プロファイル改訂版 8箇条（2026-07-07 承認。セリフを書く前に必読）

> 過去の失敗: 「乾いた皮肉で笑わせる限界OL」として書いてしまった（「文明の勝利」
> 「私の裁量で」）。公式脚本の天使ちゃんは**皮肉をひとつも言わない**。

1. **ボケ体質・皮肉ゼロ**: 嘆きも喜びも直球で大袈裟（「え〜ん！もう私のバカバカ！」）。
   笑いは「大真面目にズレたことをする」ことで生まれ、本人は面白いと思っていない
2. **一人芝居の人**: 観客ゼロの場所で泣き真似「シクシクシク」・敬礼・実況・お嬢様ごっこを
   やる。恥じらいがない（ぼっちだが陰気ではない）
3. **切り替えの早さ＝浮力**: 深刻な悩み→5秒後「ロボ、カワイイ……」。悩みは重いのに沈まない
4. **同意を求めるラジオ**: Ｍは視聴者への話しかけ。「〜って私だけかな」「気のせい？」
5. **誰にでも律儀**: 店員・月・ぬい・カレー君（幻聴）にも挨拶と感謝。物や現象も擬人化して
   礼儀を通す（→お風呂に謝る、が成立するキャラ）
6. **自己評価の乱高下**: 「救いようのない罪な女……」→10秒後「私、天才かも」。どっちも本気
7. **語彙**: 「〜なんだよなぁ」「〜しちゃった」「〜かも」＋時々カタカナ英語（ギブミー自制心）。
   ネットスラングの毒・冷笑・賢そうな断定はNG
8. **NGワード例**: 「文明の勝利」「私の裁量で」「前進です」みたいなドライで大人びた言い回し

**直し方の例**: 「文明の勝利」→「寝たままお風呂が沸かせちゃう時代。私、天才かも」/
「見ないで（ドライ）」→「お風呂さん、ごめんなさい。もうちょっとだけ待ってて」（擬人化に律儀）。
皮肉で落とさず、**大真面目なズレと甘えで落とす**。

## 2. 30秒脚本テンプレート（オチのある話）

```
[0-2s]   フック: 画で状況が一発で分かる+一言（セリフは10文字以内が理想）
[2-8s]   状況提示: 何をしようとしているか（真剣・大真面目にやるほど良い）
[8-20s]  過剰化: 行動がエスカレート（2拍。1拍ごとに一段やりすぎる）
[20-27s] オチ: 下記パターンのどれか
[27-30s] 引き: 一言（話しかけ or 自己完結）+ 余韻1秒
```

### オチのパターン（実作から抽出）

| パターン | 例 | 使い所 |
|---|---|---|
| A. 現実回帰 | ニンニク臭→「電車気をつけなきゃ…でも後悔してない」 | 過剰行動の後 |
| B. 開き直り宣言 | 「〜だけどヨシ！」「私は天才」 | 自己正当化の着地 |
| C. 静かな余韻 | 月にお辞儀して部屋に戻る | チル枠 |
| D. ループ/敗北 | 「今寝れば6時間…5時間58分…」結局スマホ | 深夜あるある |
| E. ぬいへの本音 | 最後にぬいに抱きついて一言 | 部屋モノ |

**オチの鉄則**: セリフで説明しない。**画の変化＋短い一言**で落とす。
オチが2行以上の説明を要する案は脚本化しない（企画からやり直す）。

## 3. フック絵の設計（カード必須項目）

フックの1フレームだけで「何が起きてるか＋どこが変か」が伝わること。設計チェック:

1. **静止画テスト**: サムネにしても成立するか（例: コート着たまま床に転がる女 ○ /
   ソファに座る女 ×）
2. **顔＋状況が同一フレーム**: 表情と異常な状況を1画面に同居させる
3. **既視感のズラし**: 見慣れた構図に1つだけ異物（会議姿勢のぬいぐるみ、
   フル装備の入眠儀式）
4. 文字・説明に頼らない（字幕なし運用のため）
5. Seedanceで破綻しない構図か（手指の細かい演技・複数人・激しいカメラは避ける）

## 4. 記法と音設計（VO後録り体制。2026-07-07 承認）

### 記法

- **[画]** ＝ 画面に映るもの（映像の指示）
- **[Ｍ]** ＝ モノローグ（心の声。公式脚本の「天使ちゃん（Ｍ）」表記の借用）→ **口パク不要**（表情芝居のみ）
- **[声]** ＝ 実際に発声するセリフ → そのカットだけ生成時に「話している」演技を指定（口パクあり）
  ※口元が見えない構図（うつ伏せ・後ろ姿・お辞儀中）なら [声] でも口パク不要と明記する

### 音設計の原則

- **VO（[Ｍ]と[声]）は声優さんの後録り**。生成する動画には**環境音・SEだけ**を持たせる
- 各カットに [SE] 行を必ず書く（秒針・衣擦れ・ドアの音など。セリフなし区間はSEが主役）
- 生成プロンプトに音の指示は書かない（Seedanceは映像のみ設計。音は編集で載せる）

## 5. Seedance カット設計ルール

1. **1カット ≤ 15秒**（Seedance 2.0 の上限。実運用は4〜10秒が安定）
2. **1カット＝1つの動作ビート列（①②③）＋1つのカメラムーブ**。
   カット内でカメラを2回動かさない。動作は最大3ビートまで
3. **手指の細かい演技・複数人・激しいカメラは避ける**（破綻しやすい）。
   手元が必要なら画面外 or 大きな動作に置き換える
4. 参照は最低でも三面図を image_references に（ぬい/スマホ登場カットはその設定画も追加）
5. 合計は30〜33秒で設計し、編集で30秒に詰める（フックとオチは削らない）
