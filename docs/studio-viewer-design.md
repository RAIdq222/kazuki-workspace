# 天使ちゃんスタジオ ビューアー設計「プレビュースタジオ」（v18設計・実装前）

目的: アウトプット確認(Viewer)体験の改善。参考: 上=映像・下=意図カードの2段構造
（ユーザー提示のX投稿スタイル）。「cut1から繋いでcut2のtake1/take2を切り替え再生し、
どちらが自然かを比較しやすく」がコア要件。

## コンセプト
- カード詳細(v17)=読む・編集する場所。その上に**視聴と比較に特化したフルスクリーンビューアー**を重ねる
- レビューは「見ながらその場で判断」で完結（検収OK/リテイクをビューアー内に）

## 画面構成（モバイル1カラム / PCは左プレーヤー右パネル）
1. プレーヤー（シアター表示・9:16大）
   - 再生対象は「シーケンス」= カット順に採用テイクを連結した仮想プレイリスト
     （final_url=音声マージ後を優先、無ければ生成素材）
   - タイムラインバーにカット境界マーカー（①②③…）。タップでカット頭へシーク
2. 意図パネル（プレーヤー直下・再生に同期して切替）
   - カット名・尺 / 意図（cuts[].prompt=概要） / シナリオ該当ブロック（【カットN】をパース） / 音の注記
3. テイク比較モード（本丸）
   - 対象カットのテイクをA/Bチップで並べる
   - つなぎ再生: 「前カット末尾n秒（既定2s・1〜4s調整可）→対象テイク」をワンボタン再生。
     チップ切替で同じつなぎ位置から即リプレイ（プリロード済み）
   - ←→でA/B切替・Rでリプレイ・「このテイクを採用」をその場に（adopted と同一データ）
4. モードタブ: 通し（既定）/ カット単位 / 比較
5. 検収動線: 右上に「完成にする（検収OK）」「リテイクを依頼」（リテイクは今見ているカットをプリセット）
6. 入口: ステータスヘッダー「▶ 通しで確認」（レビュー時は主ボタン）＋各カット行の「比較」

## 技術設計（スキーマ変更なし）
- 連結: video要素2枚のダブルバッファで先読み切替（ほぼgapless。MSEはPhase2）
- 同期: cuts の尺から現在カットを算出。scenario は【カットN】見出しでブロック抽出
- データ: cuts(意図) / scenario / card_jobs(cut_n, adopted, final_url) / deliverables(結合版) で充足

## フェーズ
- Phase 1: ビューアー＋意図同期＋境界シーク＋つなぎA/B比較＋採用＋検収動線
- Phase 2: 完全gapless(MSE)・PC2画面同時比較・スワイプ切替・比較履歴
