## 2. 全景ボードの建物インベントリ(layout.py 初期仮置き)

## 3. 最重要の発見: 2様式の混在(ゲート①で要確認)

ファイル名接頭辞と配色が相関する:
- **宮廷系(01,08)+城下町(05)+宮邸(13,書斎)**: 灰瓦・白漆喰・退色赤褐木部・灰切石
- **皇宮系(17,18,19,25)+客邸・顧家**: 橙(琥珀)琉璃瓦・赤壁・朱柱・金彫刻。最高格の主殿だけ濃灰瓦(紫禁城と逆転)

→ layout.py は style="court_grey" / "imperial_amber" の2マテリアルセットを最初から持つ。
「同一敷地の外郭=灰・内廷=橙」か「別複合体」かは脚本情報が必要 → **ゲート①の最優先確認事項**。
また01全景と17主殿は広場の作りが完全には一致しない(01を正として17を載せ替えるか、分離するか要承認)。

## 5. 内装の共通様式(Phase 4仕様の種)

1. 床: 大判磨き石タイル(白灰〜灰緑、ヒビ)がほぼ全室共通。顧家茶殿のみ広幅板張り → PIL 2種で全室
2. 壁: 木軸+経年白漆喰の真壁+腰板張り。梁見せ天井+回字文欄間帯が全室にある
3. 格子5種: 菱 / 方形 / 八角+方形 / 回字迷路 / 寿字円文 → lattice_panel のパターン
4. 簾(巻き上げ)が中間領域に頻出 → モデル1種で使い回し
5. 調度定番10点: 衝立(白紙/水墨)、置き行灯、灯籠柱、低卓+丸藁座布団、大絨毯(赤金唐草/黒金龍)、巻物棚、長櫃、香炉、燭台、鉢植え
6. 格の表現則: 色数を増やさず金の量で格を上げる(庶民=木+紙 → 宮邸=赤+漆喰 → 客殿=朱+金 → 祝殿=朱+金彫刻+黒漆+龍)
7. 時間帯: 朝/昼ペア(01,05)、夕(24)、夜(13,25) がそのままビューワープリセットのリファレンス

### 7.4 ゲート①②でユーザーに確認すべき事項(資料+ボード統合)
