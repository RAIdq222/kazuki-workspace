## 1. チャンネルの型（実測）

### キャラクター「天使ちゃん」の設定（動画解析から抽出）

- 20代独身OL。職場の理不尽（会議ひっくり返し・嫌味な先輩）で病む→深夜に回復する
- うさぎ耳フード付き白パーカーが定番。紫のブラウス＋黒スカート＋チョーカー
- 茶髪ロング＋ピンクインナーカラー、紫の目。脚に絆創膏（緑/青）
- ぼっち気味・感情表現が大きい・ちょっとポンコツ・自己肯定は食とささやかな幸せで回復
- 部屋: 1K、白うさぎの巨大ぬいぐるみ、パステル調。夜の街歩きシーンが多い

## 3. ミーム→天使ちゃん変換ルール

1. **文脈翻訳**: トレンドを「OLの日常 / 深夜 / ぼっち / ささやかな回復」の文脈に置き直す。
   ニュースそのものを扱わず「あるある」に落とす（例: サッカー日本戦→深夜観戦で叫んで壁ドン）
2. **キャラを壊さない**: 政治・事故・訃報・下品ネタはNG。毒はあっても自虐方向のみ
3. **飯テロに逃げない**（このシステムの目的）: 食が出てもよいが主役にしない
4. **1ショート=1ネタ**: 編集の型（docs/shorts-editing-style.md）は共通
   （フック1〜2秒→展開→オチ、16〜34秒。Seedance 制約でカットは4〜15秒×N）
5. **音源系ミームはダンス/口パク participation 型**に変換（②の型）

## 4. 企画カードのフォーマット

```
### 案N【型①〜⑤】タイトル案（チャンネル文法・絵文字なし・#深夜のドカ食い天使ちゃん）
- 元ネタ: <トレンド項目 + ソース名>
- 構成: フック(1-2s) / 展開 / オチ  ※カット割り = Seedance 4〜15秒 × N本
- 必要素材: 参照画像（立ち絵/部屋/小物）・音源
- プロンプト骨子: <Seedance用・英語>
- コスト概算: 45cr/10秒(720p,std) × カット数
```

## 5. 制作フロー（ルートB＝ゼロから生成）

1. 企画カードをユーザーが選択（or 修正指示）
2. 参照画像を用意（下記6）→ media_upload
3. `generate_video` model=seedance_2_0, aspect_ratio=9:16, image_references=立ち絵等
   でカット毎に生成 → プレビュー確認 → リテイクは最大2回
4. `explainer_video` でカット結合（無料）→ virality_predictor でフック確認（任意）
5. 完パケ（タイトル: docs/shorts-metadata-style.md のルール、定型説明文、固定タグ）

## 6. 参照資料（あると生成品質・キャラ一貫性が大きく上がる）

| 資料 | 用途 | 優先度 |
|---|---|---|
| キャラ三面図（正面/横/背面） | Seedance image_references の主軸。全カットの同一性 | ★★★ |
| 立ち絵・表情差分（喜/泣/怒/照れ/ジト目） | 感情カットの安定 | ★★★ |
| 部屋の設定画 or 美術ボード | ①生活公開型の背景一貫性 | ★★ |
| 衣装バリエ（パーカー以外・パジャマ等） | ネタの幅（休日回・就寝回） | ★★ |
| 小物設定（ぬいぐるみ・スマホ・社員証等） | 小ネタの解像度 | ★ |
| 使用可能な音源リスト（権利範囲） | ②ダンス/音源ミーム型の可否判断 | ★★ |

**フォールバック（資料なしでも可）**: 既存動画から立ち絵相当のクリーンなフレームを
抽出して image_references に使う（実績: 顔・衣装の再現はこの方法でもある程度可能）。
三面図があれば後ろ姿・横顔カットの破綻が減る。
