## 2. 時代背景・世界観（GLOBAL層の根拠）

- **タイトルの意味**: 「借屍還魂（しゃくしかんこん）」＝**死者の魂が別の死体（屍）を借りて蘇る**という
  中国の超自然概念（兵法三十六計の一。怪異譚の常套）。本話の「復活の儀」がこれにあたる。
- **時代様式**: **中国 南北朝〜初唐ごろ**を思わせる古代中国。和風・西洋・近代要素は禁。
- **世界観モチーフ**: 道教（道観＝道教寺院／道士）、**護符（呪符）・呪術・術師**、**妖怪・精霊・化け狐**、
  山岳の自然林。豪族の邸（花家）。
- **画面の基調**: 荘厳・簡素な時代劇。建築・構造・自然の骨格線が画面を主導。ファンタジー的な
  過剰装飾や巨大建造物は避ける。

> GLOBAL に常に効かせる語彙: `Chinese Northern-and-Southern-Dynasties to early-Tang period`,
> `Daoist / talisman / sorcery motifs`, `no Japanese / European / modern elements`.

## 4. シーン・場所別 設定資料（scene_profile 層の中核）

各シーンの cut範囲・担当・board・時間帯・場の設定・場面の流れ。
`✅`=構造化プロファイル有り（`runs/scene_profiles/*.json`）／`⬜`=未整備（要ボード確認）。

### ✅ 復活の儀の部屋 ｜ cut15–52（25カット）｜担当 GKV
- **board**: `SZ#6_復活の儀の部屋(夜)_R1.png` / `#6_花氏邸_全景（夜）_R2.png`
- **時間帯**: 夜（室内）
- **場所(JP/EN)**: 花家邸内の「復活の儀」の部屋 / interior ritual chamber in the Hua (花) residence
- **structures**: 木組みの柱と梁／格子の木戸・衝立／一段高い祭壇・儀式の壇／垂れ幕・幡／
  板敷きまたは石敷きの床／据え置きの燭台・灯火台
- **avoid**: 和風内装（畳・障子・襖）／西洋・近代の家具／ファンタジー的過剰装飾
- **場面**: 苗の独白（蜂のくだり）→ 儀式で身体が動かず恐怖 → 「ごらん様が蘇った」。荘厳・静的。
- 関連: 外観の引きは ✅ **花氏邸_全景**（瓦屋根・反り軒の塀／木組みの門と建屋／石段）

## 5. プロンプト3層との対応（使い方）

| 層 | 内容 | データソース |
|---|---|---|
| **GLOBAL** | 時代様式・線質・全話共通の禁則（§2） | プロンプト基盤の GLOBAL |
| **scene_profile** | 場所の語彙（structures / avoid / style）。§4 | `runs/scene_profiles/*.json` |
| **[CUT] situation** | そのカットの動き・時間帯・除去対象。§4 各場面＋ | `runs/conte_raw_ep7.csv`（action/dialogue/se/time） |

- 確認コマンド: `python -m genzu_fix.prompt show --cut N`（カット N に各層が乗るか）
- カバレッジ: `runs/scene_coverage_ep7.md`（現状 206/245カットが profile カバー、森系で84%）
- **未整備の優先**: `道観.json` / `道観_寝室.json`（計32カット）。§4 のドラフトを起点に、
  ボード `#6_山の中道教の寺院…` を確認して authoring すれば全シーンが揃う。
