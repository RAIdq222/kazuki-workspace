## 2. 絵コンテ用語（カメラワーク・編集記号）
OCRが記号・略号を誤読しないための辞書。読めた略号はこの正規形に寄せる。
| 略号/記号 | 意味 |
|---|---|
| △ | 記号（コンテ上のマーク。指示の頭などに付く） |
| F.I. | フェードイン |
| F.O. | フェードアウト |
| O.L. | オーバーラップ（ディゾルブ） |
| PAN（パン）/ Pan↑↓→ | カメラを振る |
| T.U.（トラックアップ）| 寄り（カメラを被写体へ寄せる） |
| T.B.（トラックバック）| 引き |
| FIX | カメラ固定 |
| BANK | バンク（流用カット）。例「#6のBANK」 |
| 兼用 | 流用（例「+6%兼用」） |
| BG | 背景 |
| BOOK / B① | ブック（前景の別セル。燭台・木・モヤ等） |
| IN / OUT・フレームイン | 被写体が画面に入る/出る |
| Follow | フォロー（被写体を追う） |
| ↓矢印 | 次カットへ continue（芝居が続く） |
| 縦棒｜（scene欄） | カット継続記号。scene欄に丸数字でなく縦棒だけのコマ＝直前カットの続き（同一カット）。新番号ではない。数字「1」と誤読しない |
| A.C. / AC | アクションカット（アクションつなぎ）。前カットの動き尻と次カットの頭を連続させる指定＝動きが途切れず繋がって見えるようにする。例: 蝶の動きを次カットへ連続させる |
| 全尺 | カメラ移動をカットの**全尺（全コマ）を使って**行う |
| 普通 / 普通（の速度） | 移動の**速度＝通常**。PAN/T.U.等とセットで「全尺・普通で」のように書かれる |
| じわ / じわっと | ゆっくり移動（slow PAN/T.U.）。例「↑じわPAN UP」 |
| 速い・遅い／ハヤ・オソ | 移動速度の指定 |
| 送り（上→下/左→右） | 移動方向の指定（例「TB:送り 上→下」） |

> 補足: AC は背景より芝居（動きの連続性）の指定。演出役の文脈に効く。
> **カメラ移動は「動詞(PAN/T.U./TB) ＋ 方向(↑↓→) ＋ 尺(全尺) ＋ 速度(普通/じわ)」のまとまり**で書かれる。
> 崩し字で意味不明な断片は、このまとまりの誤読を疑う（例「叙・築地で」→「全尺・普通で」）。
> 黒江さんへ: 他にも意味の分かる用語があれば追記/訂正してください。

## 3. 尺（time列）の記法
- **「秒 + コマ」表記**。例 `1+12` = 1秒12コマ、`4+12` = 4秒12コマ（24fps前提）。
- OCRは time を「整数+整数」で読み、`<秒>+<コマ>` として解釈する。`4 1/2` のような分数読みは誤り。
