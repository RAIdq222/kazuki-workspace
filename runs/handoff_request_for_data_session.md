## A. 各カットの「原図理解」（最優先・プロンプトCUT層に直結）
10カット各々、原図PNG＋絵コンテを見て次を記述（箇条書きでよい）:
1. **構図種別**: 一点 / 二点 / 三点 透視・パン・フカン/アオリ等
2. **アイレベル・画角・カメラ**: 例「ローアングル、EL=画面下1/4、広角ぎみ」
3. **主要構成物の配置と前後関係**（＝守る制約）: 何が左右・手前・奥にあるか
4. **消すもの (remove)**: 人物/動物/小道具・手書き指示・パース補助線・タップ穴 等の具体
5. **見切れ / 前景BOOK / モヤ・霧**の有無と位置
6. **time / weather の実際の付き方**: 夜・夕方・雨などが原図でどう表現/指示されているか
7. **明らかな破綻（直す対象）**: パース収束のズレ・比率の狂い・弱い構造 など気づく点

> 形式の希望: カットごとに見出し `## cut 274` を立て、上記1–7を列挙。
